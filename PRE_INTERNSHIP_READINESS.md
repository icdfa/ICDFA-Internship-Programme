## 1. Purpose and Overview

This assessment is designed to help you evaluate your readiness for the ICDFA Cybersecurity Internship Programme (CIP). The assessment consists of two parts:

1. **Self-Assessment Checklist**: A series of statements about your skills and knowledge
2. **Diagnostic Quiz**: 20 multiple-choice questions covering foundational cybersecurity concepts

This assessment is NOT a pass/fail exam. Rather, it helps you identify areas where you are strong and areas where you may need additional preparation before starting the CIP.

---

## 2. Self-Assessment Checklist

For each statement below, indicate whether you: **Strongly Agree (SA)**, **Agree (A)**, **Neutral (N)**, **Disagree (D)**, or **Strongly Disagree (SD)**.

### 2.1 Basic Computer Literacy

- [ ] I am comfortable using a computer for everyday tasks (email, web browsing, file management)
- [ ] I understand basic file system concepts (folders, files, file paths)
- [ ] I can install and uninstall software on my computer
- [ ] I understand the difference between hardware and software
- [ ] I am familiar with common operating systems (Windows, macOS, Linux)
- [ ] I can troubleshoot basic computer problems (restarting, clearing cache, etc.)
- [ ] I understand what an IP address is and why it's important
- [ ] I know how to check my internet connection speed
- [ ] I am comfortable learning new software and tools
- [ ] I have experience using virtual machines or virtualisation software

### 2.2 Networking Fundamentals

- [ ] I understand the basic concept of computer networks
- [ ] I know what the internet is and how it works at a basic level
- [ ] I understand the difference between LANs and WANs
- [ ] I am familiar with the concept of IP addresses and subnets
- [ ] I know what a firewall is and why it's important
- [ ] I understand the basic concept of ports and protocols
- [ ] I have heard of common protocols like HTTP, HTTPS, DNS, and FTP
- [ ] I understand the concept of client-server architecture

### 2.3 Command Line Familiarity

- [ ] I have used a command-line interface (terminal or command prompt) before
- [ ] I am comfortable navigating the file system using command-line commands
- [ ] I can create, move, and delete files and folders using the command line
- [ ] I understand basic command-line syntax and how to use flags/options
- [ ] I have used commands like ls, cd, pwd, mkdir, rm, or similar
- [ ] I am comfortable reading command-line output and error messages
- [ ] I understand the concept of file permissions and ownership
- [ ] I am willing to learn more advanced command-line skills

### 2.4 Security Awareness

- [ ] I understand the basic concept of cybersecurity
- [ ] I know what malware is and how it can affect computers
- [ ] I understand the importance of strong passwords
- [ ] I am familiar with the concept of phishing and social engineering
- [ ] I know what encryption is and why it's important
- [ ] I understand the concept of authentication and authorisation
- [ ] I am interested in learning more about cybersecurity threats and defences

## 4. Scoring Guide

### 4.1 Self-Assessment Checklist

Count your responses for each section:

- **Strongly Agree (SA)**: 5 points
- **Agree (A)**: 4 points
- **Neutral (N)**: 3 points
- **Disagree (D)**: 2 points
- **Strongly Disagree (SD)**: 1 point

Calculate your score for each section:

- **Basic Computer Literacy**: _____ / 50 points
- **Networking Fundamentals**: _____ / 40 points
- **Command Line Familiarity**: _____ / 40 points
- **Security Awareness**: _____ / 35 points

**Total Self-Assessment Score**: _____ / 165 points

### 4.2 Diagnostic Quiz

Count the number of correct answers: _____ / 20

**Quiz Score**: _____ %

### 4.3 Overall Readiness Score

**Overall Score** = (Self-Assessment Score / 165) × 50 + (Quiz Score / 100) × 50

**Overall Score**: _____ / 100

### 4.4 Readiness Levels

- **Ready (80-100)**: You are well-prepared for the CIP. You have strong foundational knowledge and skills. You can proceed directly to the CIP.
- **Partially Ready (60-79)**: You have some foundational knowledge but may benefit from additional preparation. You may be offered conditional admission with the option to complete a bridge course.
- **Not Yet Ready (Below 60)**: You may not be ready for the CIP at this time. We recommend completing additional preparation before applying. See Section 5 for recommended resources.

## 5. Recommended Preparation Resources

Based on your assessment results, here are recommended resources to strengthen your skills:

### 5.1 For Basic Computer Literacy

- **YouTube Channels**: Professor Messer, Crash Course Computer Science
- **Online Courses**: Coursera - "Fundamentals of Computing", Khan Academy - "Computing"
- **Books**: "The Basics of Computer Science" by Paul Tobin
- **Practice**: Spend time using different operating systems and software

### 5.2 For Networking Fundamentals

- **YouTube Channels**: Professor Messer, Cisco Networking Academy
- **Online Courses**: Coursera - "Introduction to Networking", Udemy - "Networking Basics"
- **Books**: "CompTIA Network+ Study Guide" by Todd Lammle
- **Practice**: Use network simulation tools like Cisco Packet Tracer

### 5.3 For Command Line Familiarity

- **YouTube Channels**: Traversy Media, The Linux Foundation
- **Online Courses**: Linux Academy, Codecademy - "Learn the Command Line"
- **Books**: "The Linux Command Line" by William Shotts
- **Practice**: Use Linux or macOS terminal daily; complete hands-on exercises

### 5.4 For Security Awareness

- **YouTube Channels**: Professor Messer, John Hammond
- **Online Courses**: Coursera - "Introduction to Cybersecurity", edX - "Cybersecurity Fundamentals"
- **Books**: "The Basics of Cybersecurity" by Paul Tobin
- **Practice**: Research current cybersecurity news and threats
